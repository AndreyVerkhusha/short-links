## Запуск проекта

```bash
git clone https://github.com/AndreyVerkhusha/short-links.git
cd short-links
composer install
php artisan migrate
php artisan serve
composer pint // опционально, добавил быстрый скрипт. 
```
В проект уже включен файл `.env`, обычно так не делается, добавил просто для вашего удобства.

## Реализованный функционал:

### Выполненные требования ТЗ

- Регистрация и аутентификация пользователей
- Админ-панель на Filament v3
- Каждый пользователь имеет доступ только к своим ссылкам и их статистике.
- Создание коротких ссылок
- Переход по короткой ссылке с перенаправлением на оригинальный URL
- Запись каждого перехода в статистику
- Просмотр списка созданных ссылок
- Удаление ссылок
- Просмотр статистики по каждой ссылке
- Сохранение ip адреса посетителя
- Сохранение даты и времени каждого перехода

### Дополнительно

- Использовал построение по сервисной архитектуре Service Layer
- Логика вынесена из контроллеров в сервисы, контроллеры остаются тонкими
- Для работы с бд используются репозитории
- Добаивил в composer скрипт на быстрый запуск pint для форматирования
